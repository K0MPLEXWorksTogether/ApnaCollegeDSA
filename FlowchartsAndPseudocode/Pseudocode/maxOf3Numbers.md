# Pseudocode For Maximum Of 3 Numbers

## Input
Three numbers, a, b and c.

## Output
The maximum among the three numbers.

## Pseudocode:
1. Start.
2. Read numbers a, b and c.
3. ```
    if a > b do
        if a > c do
            Write a
        else 
            Write c
    else
        if b > c do
            Write b
        else
            Write c
   ```
4. Exit.
