# Pseudocode To Find If A Number Is Prime

## Input
A number n.

## Output
Prime, if the number is prime, else Not Prime

# Pseudocode

1. Start.
2. Read the number n.
3. Initialise d = 2
4. ```
   while d < n do
        if n % d == 0 do
            Write "Not Prime"
            Exit.
        else
            d = d + 1
   ```
5. Write "Prime"
6. Exit.
