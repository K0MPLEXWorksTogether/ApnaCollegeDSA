# Pseudocode To Print The Average Of 25 Scores

## Input
The 25 scores.

## Output
The average of the 25 scores.

## Pseudocode

1. Start.
2. Initailise sum = 0, count = 0
3. ```
    while count < 0 do
        Read score
        sum = sum + score
        count = count + 1
    ```
4. Calculate avg = sum / 25
5. Write avg.
6. Exit.
